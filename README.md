# Det Battle

行列式を速く正確に計算して競う、オフライン対戦ゲームです。

## 起動方法

`index.html` をブラウザで開きます。ES Modulesを使用しているため、直接開いて動作しない場合はローカルサーバーから配信してください。

```sh
python -m http.server 8000
```

起動後、ブラウザで `http://localhost:8000` を開きます。

## 現在の状態

第一版が完成しています。同じ行列を2人で同時に解き、共有入力欄から回答します。3ラウンド終了時の合計点が低いプレイヤーの勝利です。

## 得点方法

1ラウンドの得点は、正解までの経過秒数（1秒未満切り捨て）と、各誤答と正解との差の絶対値の合計です。
