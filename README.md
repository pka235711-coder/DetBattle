# Det Battle

行列式を速く正確に計算して競う、オフライン対戦ゲームです。

## 起動方法

`index.html` をブラウザで開きます。ES Modulesを使用しているため、直接開いて動作しない場合はローカルサーバーから配信してください。

```sh
python -m http.server 8000
```

起動後、ブラウザで `http://localhost:8000` を開きます。

## 現在の状態

同じ行列を2人で同時に解き、共有入力欄から回答します。設定したラウンド数の終了時に、合計点が低いプレイヤーの勝利です。ラウンド数の初期値は3です。

## 得点方法

1ラウンドの得点は、`正解までの経過秒数（1秒未満切り捨て）+ 誤差合計 × 設定倍率`です。倍率は0.00〜99.99の範囲で設定でき、初期値は1.00です。

得点の内訳は、両プレイヤーが正解してラウンドが終了するまで表示されません。
